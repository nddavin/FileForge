# FileForge Privacy Policy

**Last Updated:** January 2024

## 1. Introduction

FileForge ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our enterprise file processing platform.

By using FileForge, you agree to the collection and use of information in accordance with this policy.

## 2. Information We Collect

### 2.1 Personal Information

| Category | Examples | Purpose |
|----------|----------|---------|
| Identity Data | Name, email, organization | Account management |
| Contact Data | Email address, phone | Communication |
| Credentials | Passwords, API keys | Authentication |
| Usage Data | Login times, features used | Service improvement |

### 2.2 Church & Organization Data

| Category | Examples | Purpose |
|----------|----------|---------|

| Sermon Data | Audio/video files, transcripts | Processing services |

### 2.3 Automatically Collected Data

| Category | Examples | Purpose |
|----------|----------|---------|
| Technical Data | IP address, browser type, device | Security, diagnostics |
| Usage Metrics | Features accessed, processing times | Service improvement |
| Logs | Operation history, errors | Troubleshooting |

## 3. How We Use Your Information

### 3.1 Core Processing

- **Sermon Processing**: Transcoding, metadata extraction, speaker identification
- **File Organization**: Smart sorting, categorization, retrieval
- **Search & Discovery**: Full-text indexing, filtering

### 3.2 Service Improvement

- Analyzing usage patterns to improve features
- Developing new capabilities based on anonymized data
- Performance monitoring and optimization

### 3.3 Communication

- Account notifications and updates
- Security alerts and announcements
- Customer support responses

## 4. Data Sharing & Disclosure

### 4.1 We Do NOT Sell Your Data

Your church data, sermon files, and member information are never sold to third parties.

### 4.2 Service Providers

We share data with trusted service providers who assist in operating our platform:

| Provider | Purpose | Data Shared |
|----------|---------|-------------|
| Cloud Hosting | Infrastructure | Processing data |
| Payment Processors | Billing | Transaction data |
| Analytics | Usage metrics | Aggregated data |

### 4.3 Legal Requirements

We may disclose data when required by law:
- Court orders or subpoenas
- Regulatory investigations
- Compliance with legal obligations

### 4.4 Organizational Changes

In the event of a merger, acquisition, or sale, data may be transferred as a business asset.

## 5. Data Protection

### 5.1 Encryption

| State | Method | Details |
|-------|--------|---------|
| In Transit | TLS 1.2+ | All network communication |
| At Rest | AES-256 | File storage encryption |
| Processing | Memory encryption | Runtime data protection |

### 5.2 Access Controls

- Role-based access control (RBAC)
- Multi-factor authentication (MFA)
- Session management and timeouts

### 5.3 Data Retention

| Data Type | Retention | Policy |
|-----------|-----------|--------|
| Sermon Files | Configurable | Per organizational policy |
| Metadata | Duration of account | Plus 30 days |
| Logs | 7 years | Compliance requirement |
| Backups | 90 days | Archive rotation |
